# 云函数目录

## 说明

此目录用于存放微信云开发的云函数。

## 目录结构

```
cloudfunctions/
├── common/              # 公共函数
│   ├── middleware.js    # 租户上下文注入中间件
│   ├── auth.js          # 认证工具函数
│   ├── response.js      # 统一响应格式
│   └── logger.js        # 日志记录
├── auth/                # 认证相关云函数
│   ├── index.js
│   └── package.json
└── ...
```

## 创建云函数

在微信开发者工具中：
1. 右键点击 `cloudfunctions` 目录
2. 选择"新建云函数"
3. 输入云函数名称

## 部署云函数

在微信开发者工具中：
1. 右键点击云函数目录
2. 选择"上传并部署：云端安装依赖"
